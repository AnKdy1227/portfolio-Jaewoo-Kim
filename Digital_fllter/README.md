# Digital Filter Implementation

간단한 디지털 필터들을 C로 구현한 프로젝트입니다.

## 📌 Features
- LPF (Low Pass Filter)
- HPF (High Pass Filter)
- BPF (Band Pass Filter)
- BSF (Band Stop Filter)
- APF (All Pass Filter)

## 🧠 Description
기본적인 신호 처리(DSP) 개념을 바탕으로  
여러 종류의 필터를 직접 구현하였습니다.  
각 필터의 동작을 코드로 확인할 수 있습니다.

## 🛠️ Tech
- Language: C

## 📁 Structure
- main.c : 프로그램 실행 및 테스트
- filter.c : 필터 구현
- filter.h : 필터 헤더

## ▶️ Usage
gcc main.c filter.c -o filter
./filter
